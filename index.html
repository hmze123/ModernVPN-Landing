CORE ENGINES

كافة بروتوكولات الأنفاق المدعومة

Xray (V2Ray)

دعم VLESS و VMess عبر عمليات منفصلة لضمان استقرار الأداء 100%.

SSH Tunneling

دعم Direct, SSL Proxy, و Payload Custom لتخطي الجدران النارية.

OpenVPN

دمج المحرك الرسمي لاتصالات عالمية مشفرة ومستقرة جداً.

DNSTT Stealth

نقل البيانات عبر طلبات DNS، الحل الأخير عند إغلاق كافة المنافذ.

UDP Hysteria

بروتوكول UDP فائق السرعة مخصص للألعاب والبث المباشر (Ping منخفض).

Psiphon Tunnel

محرك القفز بين البروتوكولات لتجاوز الرقابة الشديدة آلياً.

Shadowsocks

بروتوكول تشفير خفيف وسريع جداً لتخطي الحجب الجغرافي.

Desync TCP

نظام متطور لتجزئة الحزم وتضليل أنظمة الرقابة العميقة (DPI).

NETWORK LABORATORY

مختبر الأدوات المتكامل

Host Checker

تحليل المضيفين واستخراج الهيدرز لاختبار ثغرات الشبكة وبناء Payload.

Magic IP Hunter

البحث الآلي عن الآي بي المتوافق مع متطلبات شبكتك لفتح الاتصال.

Port Scanner

كشف المنافذ المفتوحة وتحليل الخدمات النشطة في السيرفرات البعيدة.

Ping Monitor

مراقبة لحظية لزمن الاستجابة مع رسم بياني لمستوى استقرار الاتصال.

DNS Leak Test

التأكد التام من عدم تسريب هويتك الحقيقية عبر طلبات الـ DNS.

VPN Hotspot

مشاركة اتصال الـ VPN الخاص بك مع الأجهزة الأخرى عبر نقطة الاتصال.

Subdomain Finder

أداة احترافية للبحث عن النطاقات الفرعية المفتوحة داخل الشبكات.

IP Info

عرض تفاصيل دقيقة حول عنوان IP الخاص بك وموقع السيرفر المتصل به.
